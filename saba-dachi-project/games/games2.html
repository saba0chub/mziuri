FACTORIO

Factorio is a game all about building and managing automated
factories on an alien planet. You start by manually gathering resources
like iron, copper, and coal, but the main fun is in designing efficient production
lines that turn raw materials into increasingly complex products.

Project Zomboid

Project Zomboid is an open-world survival role-playing game (RPG) set in a zombie apocalypse.
The core of the game revolves around surviving as long as possible in a persistent,
hostile environment filled with zombies.
